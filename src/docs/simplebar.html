Simplebar

Plugin homepage: https://github.com/Grsmto/simplebar

SimpleBar does only one thing: replace the browser's default scrollbar with a custom CSS-styled one without losing performances.

Basic setup

Add "data-pixr-simplebar" to the element you want to output the simplebar on. You will also need to set a fixed height on a parent element. We have created the CSS class "simplebar-wrapper" with a fixed height for this but any CSS elem with a fixed height will do.

Template integration

You can find our JS setup here: src/assets/js/components/simplebar
