Sticky

Plugin homepage: https://rgalus.github.io/sticky-js/

Sticky-js is a library for sticky elements written in vanilla javascript. With this library you can easily set sticky elements on your website. It's also responsive.

{{> gallery/images-stacked }}

Basic setup

Add the CSS class "sticky" to the element you want to make sticky. Then add "data-sticky-container" to parent of sticky elem.

Template integration

You can find our JS setup here: src/assets/js/components/sticky
