Customising template

Let's go through the various ways in which you may want to customise the template:

  • Customise CSS/SASS
  • Customise Favicon Image
  • Customise Font
  • Customise Images/Video
  • Edit/Create HTML Pages

Customise CSS/SASS

All of the template's source CSS/SASS files are kept inside the template's assets folder. Navigate to src/assets/scss. Open up theme.scss with your editor.

This is the main entry point for all other SASS/CSS files.

We suggest that you do not write custom code directly into theme.scss. Instead, import your SASS/CSS file into this file. That way you keep your CSS tidy and well-organised.

If you want to customise Bootstrap's variables, please open up: src/assets/scss/abstracts/_sass-variables.scss

You can read more about customising Bootstrap's default SASS here: https://getbootstrap.com/docs/5.0/customize/sass/

If you only want to write custom CSS, then we have setup a file for you. Open src/assets/scss/custom.scss. Add your code, save file.

Customise Favicon Image

Create your new favicon and name it "favicon.ico". Navigate to src/assets/favicon. Replace the current favicon file with your new favicon. You may have to clear your browser's cache to see the updated favicon. Customise Font

There are different ways to add a custom font to a website. For the purpose of this helpdoc article, let's assume you have generated a @font-face kit. Navigate to src/assets/fonts and delete the existing fonts. Place your custom font files into this folder. Then in your code editor, open up file: src/assets/scss/base/_base.scss and update the code accordingly. Not sure about this step? Read more about what webfont are here: https://developer.mozilla.org/en-US/docs/Learn/CSS/Styling_text/Web_fonts

For the final step, in your code editor, open up the file: src/assets/scss/abstract/_css-vars.scss and update the value for "--bs-font-..." and "--theme-font-headings". The "-bs-font-" value will either end "sans-serif" or "serif", depending on what font type we used for your template. In most cases our templates only use a single font for both headings and body text, but there may be different fonts per element, or you may want different fonts for these elements. In that case, only change the relevant CSS property instead of updating both.

Customise Images/Video

To update images, navigate to src/assets/images. Place your custom images here.

To update videos, navigate to src/assets/videos. Place your custom videos here.

Edit/Create HTML Pages

To create a new page, navigate in your code editor to: src/html. To make it easier to get the correct HTML in place, clone an existing page (we have created a "blank.html" for this purpose, but you can use any existing page). Rename the newly-created file to whatever URL you require. And that's it. You are now free to open the new page with your code editor, make your changes, and then save the file.

You will likely have to restart Webpack when you create a new file.

To edit an existing HTML file, navigate to the same folder, open the file you wish to edit, make your changes, and save the file. Webpack will automatically refresh your browser with the updated code.
