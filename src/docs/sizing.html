Sizing

We have extended the default Bootstrap utilities for sizing to give you greater control over elements that require either a fixed width or height (or both).

Fixed width & height options

You can assign a fixed width to an element by using f-w-${value}. Replace "${value}" with a value listed below, eg "f-w-32"

The same applies for fixed height, but use class "f-h-32".

f-w-32 f-h-32
f-w-40 f-h-40
f-w-60 f-h-60
