Bootstrap default utilities

Bootstrap already has a comprehensive section in their helpdocs detailing all available utilities. Our helpdocs do not repeat their information. Instead, we focus on the custom utilities our template provides. Please start by reading through Bootstrap's Utility section of their helpdocs: https://getbootstrap.com/docs/5.0/utilities/api/

All core Bootstrap utilities are available in our template.

Bootstrap Utility API

We have setup a SCSS file to override and extend Bootstrap's utility classes through their API. You can find it here: src/assets/scss/utilities/_bootstrap-api-utilities.scss. Please refer to the Bootstrap documentation for a detailed overview of how to extend their utility classes using their API: https://getbootstrap.com/docs/5.0/utilities/api/
