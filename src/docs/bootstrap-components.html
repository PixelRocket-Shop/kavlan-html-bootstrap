Bootstrap components

All of the default Bootstrap components are available in our template. Instead of repeating the help documentation that Bootstrap already have, please read through their component section for a full overview of what components are available and how to implement them: https://getbootstrap.com/docs/5.0/components/accordion/

The components section of our helpdocs will focus on the custom components our template offers.
