Basic setup

Add the HTML elements that swiper requires. You can find those in the example below, or a better option is to grab the HTML code from the swiper.js demo page: https://swiperjs.com/demos

You then initialise the Swiper by passing in valid JSON into "data-options" through the "swiper-container" element. Be sure to use single quotes, not double quotes. You can also initialise the slider by writing custom JS - please refer to the swiper documentation for examples of this method.
