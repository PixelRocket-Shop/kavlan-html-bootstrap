Buttons

Default buttons, faded buttons, chunky buttons, and skewed buttons available. All theme colours and sizes available. Bootstrap also offers control of button size and border radius.

Default buttons

Faded buttons (custom)

Use CSS class "btn-${color}-faded" for faded variants

Chunky buttons (custom)

Use CSS class "btn-${color}-chunky" for chunky variants

Skewed buttons (custom)

Add SPAN tag and CSS class of "btn-skew-${direction}" to skew button left or right.
