Requirements

If you do not intend to work with the template source code (and that means you will not be compiling it or running the Webpack dev server), you do not need to install anything. You can simply navigate to the dist folder and start editing the files.

Be warned: if you edit the files inside the dist folder and then decide to recompile the template using Webpack, you will lose any changes made to files inside the dist folder. This is because when Webpack compiles the template, it places all files inside the dist folder, thereby overwriting the current files.

Node

Most, if not all developers will be editing the source code and will also be running Webpack to compile the template. If that's the case, then ensure that you have Node.js installed. You can download it from here: https://nodejs.org/en/download/

That's it. That's all you need to have pre-installed. You can now open the main template folder with your command line!
